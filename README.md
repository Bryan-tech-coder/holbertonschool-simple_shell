
# SIMPLE SHELL

Hola, bienvenido a nuestro proyecto de * * simple shell* *, llamado 'hsh' .
Este proyecto consiste en crear un **interprete de comandos ** que funcionen como '/bir

## objetivo
-comprender como funciona un shell desde cero.
-aprender a crear procesos, manejar senales y ejecutar programas usando **system cal
-practicar buenas practicas en C siguiendo **Betty style** .
Entener como usar variables de entorno y el PATH para ejecutar programas.


## Funcionalidades
-Ejecutar comandos como '/bin/ls', 'pwd', 'echo', etc.
-Modo interactivo (tipo terminal) y modo no interactivo (por pipes o archivos).
-manejar errores como '/bin/sh', mostrando mensajes con el nombre del programa 
-sin memory leaks y codigo limpio segun los estandares del proyecto .
-permite usar redireccion de entrada y salida basicas.

## heramientas usadas
-lenguaje: **c**
-compilador: 'gcc' en ubuntu 20.04 LTS
-Editor: 'vim' /'vi' / 'emacs'
-control ve versiones: **github**
