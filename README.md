## simple shell project

Hello!
Welcome to our simple shell project, called "hsh".

this project consists of creating a command-line interprete that works 
like/bin/sh

Objective
-understand how a shell works from scratch
-learn how to create processes, handle signals, and execute programs using system calls.
-practice good coding practices in c following the Betty style.
-understand how to use environment variables and the PATH to execute programs.

Features 
-Executes commands such as /bin/ls/pwd,echo,ect.
-Interactive mode (terminal-like) and non-interactive mode (via pipes or files)
-Handles errors such as/bin/sh,displaying messages with the program name.
-No memory leaks and clean code according to the project standards
-Allows basic input and output readirection

Tools Used
-language:C
-compiler:gcc on Ubuntu 20.04 LTS
-Editor: Vim/vi/emacs
-version control: Github
